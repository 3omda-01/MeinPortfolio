# MeinPortfolio
this is a simple terminal-style portfolio i made using just html, css, and javascript.

i wanted something different from the usual portfolio websites, so i went with a linux terminal vibe (arch + hyprland type shi). everything works with commands, just like a real terminal.

## what it does

basically you type commands and it shows info about me.

available commands:

- `whoami` → info about me  
- `education` → my studies  
- `skills` → what i use / know  
- `roles` → experience & positions  
- `contact` → how to reach me  
- `links` → github / linkedin  
- `help` → list commands  
- `clear` → clears the terminal  

## why i made it

i didn’t want a basic portfolio. this felt more "me", especially since i use linux daily.

also it was a fun way to practice js and build something clean without frameworks.

## tech used

- html
- css
- javascript (vanilla)

no libraries, no frameworks.
